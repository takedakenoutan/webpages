AccessRestriction

AccessRestrictionとは？

AccessRestrictionはその名の通りアクセスを制限するAPIです。
これを使えば、設定したページが読み込まれる前に
ダイアログを出してIDとPasswordを要求します。
ここでIDとPasswordが一致すればページが読み込まれます。
一致しなければ、設定したページに飛ばすことができます

設定方法

まず、アクセスを制限したいページのhtmlファイルのhead内に
<AccessR id= value1 pass= value2 href= link/>
と記述してください。
value1にはIDを、value2にはPasswordを、
linkには一致しなかった場合に飛ばすLinkを
すべて文字列の状態で記述してください。("で囲む)
次に
<script src= "https://takedakenoutan.github.io/webpages/
Release/AccessRestriction.js"></script>
を先程のAccessRタグよりも後ろに記述してください。
